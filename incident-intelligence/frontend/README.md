# IntelResponse – Frontend

React + TypeScript frontend for the IntelResponse incident intelligence platform, built with Vite.

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```

## Demo Flow
1. Dashboard overview with incident counts and risk summary
2. Incident list with filtering and sorting
3. Incident detail with extracted IOCs and risk score
4. Similar incident recommendations
